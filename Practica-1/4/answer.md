Operadores aritméticos

- '+' → suma
- '-' → resta
- '*' → multiplicación
- '/' → división
- '%' → módulo (resto entero)
- '++' → incremento (pre y post)
- '--' → decremento (pre y post)
- '+' (unario) → identidad
- '-' (unario) → cambio de signo


2) Operadores relacionales ( Devuelven 1 (verdadero) o 0 (falso) ):

- '==' → igual
- '!=' → distinto
- '>' → mayor
- '<' → menor
- '>=' → mayor o igual
- '<=' → menor o igual

3) Operadores lógicos ( Trabajan con valores booleanos (0 o distinto de 0) ):

- '&&' → AND lógico
- '||' → OR lógico
- '!' → NOT lógico

4) Operadores de bits (Operan bit a bit):

- '&' → AND binario
- '|' → OR binario
- '^' → XOR binario
- '~' → NOT binario
- '<<' → desplazamiento izquierda
- '>>' → desplazamiento derecha

5) Operadores de asignación:

- '=' → asignación simple
- '+=' → suma y asigna
- '-=' → resta y asigna
- '*=' → multiplica y asigna
- '/=' → divide y asigna
- '%=' → módulo y asigna
- '&=' → AND y asigna
- '|=' → OR y asigna
- '^=' → XOR y asigna
- '<<=' → desplaza izquierda y asigna
- '>>=' → desplaza derecha y asigna

    condición ? valor_si_true : valor_si_false
